# 2b2tmuseum World Downloads
This is a WIP. Some things may or may not be available.

## Contributing

1. Fork this repo
2. Add your world download(s) to their own directories in the appropriate year folder.
3. Only add the following file structures to git:
      * `SomeBase/level.dat` (file)
      * `SomeBase/region` (directory)
      * `SomeBase/entities` (directory - this dir may not exist for old world downloads)
      * `SomeBase/data` (directory - only if there are map_xxx.dat files present)
4. Make a pull request, write a quick description for what you added. Try to add the year the world download was captured, and if the world download is a download of a download, let us know where you ripped it from.
